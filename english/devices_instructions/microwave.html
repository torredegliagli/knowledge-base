Microwave instructions

Tips

  • Do not use plastic containers, even if microwave-safe, when using the grill or combi grill function.
  • Use only heat-resistant utensils for the grill function.
  • Do not touch the grill when it is hot.
  • Do not use the microwave to cook or reheat whole eggs, with or without shell, as they may explode.
  • Liquids heated in the microwave can exceed the boiling point without apparent bubbling. Let them rest briefly and stir before removing.
  • Do not use the oven without the turntable and the turntable support in position.
  • Do not turn on the oven when it is empty.

To avoid overheating objects

  • Do not use narrow-necked containers to heat liquids.
  • Always check the temperature of food for children before serving.
  • Always use potholders or oven mitts when handling hot containers.

Control panel description

Microwave mode

  • Press the button with the microwave icon.
  • Select the desired power by pressing the same button multiple times or by turning the knob.
  • Set the cooking time by turning the knob.
  • Press the start button to begin cooking.

Grill mode

  • Press the button with the grill icon.
  • Set the cooking time by turning the knob.
  • Press the start button to begin cooking.

Combi grill mode

  • Press the combi grill button.
  • Select the desired cooking type by pressing the button multiple times or by turning the knob.
  • Set the cooking time by turning the knob.
  • Press the start button to begin cooking.

Automatic defrost

  • Press the automatic defrost button.
  • Select the food category (meat, vegetables, fish, poultry) by pressing the button multiple times or by turning the knob.
  • Set the food weight by turning the knob.
  • Press start to begin the function.

Maintenance and cleaning

  • Regularly clean the oven, removing all food residues.
  • Use a soft cloth and mild detergent for the interior and exterior surfaces.
  • Do not use steam appliances to clean the oven.
  • Clean the turntable and turntable support regularly.
