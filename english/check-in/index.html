Check-in instructions

Self Check-in (via Ok Home System)

  • Receive Access: You will receive a link to access a web app and a backup code for offline use.
  • Login: Use the provided link to log into the web app.
  • Door Control: After logging in, you will see icons for three doors: Palazzo Door, Pianerottolo Door, and Entrance Door.
  • Open Doors: Click on the corresponding icon to open the desired door.
  • Notes: The Palazzo Door takes 2-3 seconds to respond after clicking on the smartphone. Do not click too quickly between the Pianerottolo Door and the Entrance Door. Use the app in the proximity of the door only.

Offline Check-in (if the Ok Home System is not available)

  • Palazzo Door: Ring the "Nardoni" doorbell X times. This will open the Palazzo Door.
  • Access Keys: Once on the 6th floor, use the numeric keypad to open the key box near the Pianerottolo Door.
  • Pianerottolo Door: Use the keys from the key box to open the Pianerottolo Door. Return the keys to the key box afterward.
  • Entrance Door: Use the second numeric keypad and enter the Ok Home code provided at the time of booking to open the Entrance Door.

Important Notes

  • The manual method involves a few extra steps, while the web app is more intuitive and quicker, requiring only a tap on the screen to open the corresponding door. Please always prefer the use of the web app over the offline method.
  • In the event of any problem when checking in or when using the Ok Home app, please contact Alessandro, the host, on these numbers: +44 07933289746 (UK) or +39 3333757637 (IT). Thank you!
