Istruzioni per il check-in

Check-in automatico (tramite il sistema Ok Home)

  • Ricevi l'accesso: Riceverai un link per accedere a un'app web e un codice di backup per l'uso offline.
  • Accedi: Utilizza il link fornito per accedere all'app web.
  • Controllo delle porte: Dopo aver effettuato l'accesso per la prima volta, vedrai una pagina di benvenuto.
  • Quindi ti verrà chiesto di inserire il tuo numero di telefono
  • Atterrerai finalmente sulla pagina principale dell'app. Clicca su "Apri porta" quando sei in prossimità della porta da aprire.
  • Vedrai icone per tre porte: APRI EDIFICIO, APRI PORTA e APRI APPARTAMENTO.
  • APRI EDIFICIO: Apre la porta sulla strada.
  • APRI PORTA: Apre la porta al sesto piano (cerca il campanello Nardoni).
  • APRI APPARTAMENTO: Apre la porta dell'appartamento.
  • Apri le porte: Clicca sull'icona corrispondente per aprire la porta desiderata.
  • Note: La porta APRI EDIFICIO impiega 2-3 secondi per rispondere dopo aver cliccato sullo smartphone. Non cliccare troppo rapidamente tra i pulsanti APRI PORTA e APRI APPARTAMENTO. Utilizza l'app solo in prossimità della porta.

Check-in offline (se il sistema Ok Home non è disponibile)

  • PORTA DELL'EDIFICIO: Suona il campanello "Nardoni" X volte. Questo aprirà la PORTA DELL'EDIFICIO sulla strada.
  • Accesso alle chiavi: Una volta al 6º piano, utilizza il tastierino numerico per aprire la cassetta delle chiavi vicino alla porta con il campanello Nardoni.
  • APRI PORTA: Utilizza le chiavi dalla cassetta delle chiavi per aprire la Porta del Pianerottolo. Ritorna le chiavi nella cassetta delle chiavi successivamente.
  • PORTA DELL'APPARTAMENTO: Utilizza il secondo tastierino numerico e inserisci il codice Ok Home che appare sulla pagina delle porte dell'app web per aprire la PORTA DELL'APPARTAMENTO.

Note importanti

  • Il metodo manuale comporta qualche passaggio in più, mentre l'app web è più intuitiva e veloce, richiedendo solo un tocco sullo schermo per aprire la porta corrispondente. Si prega di preferire sempre l'uso dell'app web rispetto al metodo offline.
