Istruzioni Ok Home

Check-in automatico (tramite il sistema Ok Home)

  • Dettagli per accedere: Riceverai un'email con un link e le credenziali per accedere a una web app, e un codice di backup per l'uso offline.
  • Accedi: Usa il link e i dettagli forniti per accedere alla web app.


  • Controllo delle porte: Dopo aver effettuato l'accesso per la prima volta, vedrai una pagina di benvenuto.


  • Ti verrà quindi chiesto di inserire il tuo numero di telefono


  • Dopo aver iserito il numero, entrerai nella home page della web app. Clicca su "Apri Porte" quando sei in prossimità della porta da aprire.


  • Vedrai un'icona per ciascuna porta da aprire: PORTA PALAZZO, PORTA PIANEROTTOLO e APRI APPARTAMENTO.


  • PORTA PALAZZO: Apre la porta sulla strada.
  • PORTA PIANEROTTOLO: Apre la porta al sesto piano (cerca il campanello Nardoni).
  • APRI APPARTAMENTO: Apre la porta dell'appartamento.
  • Clicca sull'icona corrispondente per aprire la porta desiderata.
  • Note: La PORTA PALAZZO impiega 2-3 secondi per rispondere al click sullo smartphone. Non cliccare troppo rapidamente tra i pulsanti PORTA PIANEROTTOLO e APRI APPARTAMENTO. Utilizza l'app solo in prossimità della porta.

Check-in offline (se il sistema Ok Home non è disponibile)

  • PORTA PALAZZO: Suona il campanello "Nardoni" X volte. Questo aprirà la PORTA PALAZZO sulla strada.
  • Accesso alle chiavi: Una volta al 6º piano, utilizza il tastierino numerico per aprire la cassetta delle chiavi vicino alla porta con il campanello Nardoni.
  • PORTA PIANEROTTOLO: Utilizza le chiavi dalla cassetta delle chiavi per aprire la PORTA PIANEROTTOLO. Ritorna le chiavi nella cassetta delle chiavi successivamente.
  • APRI APPARTAMENTO: Utilizza il secondo tastierino numerico e inserisci il codice di accesso che appare sulla pagina Apri Porte nella web app per APRIRE L'APPARTAMENTO.

Note importanti

  • Il metodo manuale comporta qualche passaggio in più, mentre la web app è più intuitiva e veloce, richiedendo solo un tocco sullo schermo per aprire la porta corrispondente. Si prega di preferire sempre l'uso della web app rispetto al metodo offline.
