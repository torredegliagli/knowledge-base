Instrucciones microondas

Sugerencias

  • No usar recipientes de plástico, incluso si son aptos para microondas, cuando se utilice la función grill o combi grill.
  • Usar solo utensilios resistentes al calor para la función grill.
  • No tocar la parrilla cuando esté caliente.
  • No usar el microondas para cocinar o calentar huevos enteros, con o sin cáscara, ya que podrían explotar.
  • Los líquidos calentados en el microondas pueden superar el punto de ebullición sin formar burbujas visibles. Deje reposar un breve momento y mezcle antes de retirarlos.
  • No usar el microondas sin el plato giratorio y el soporte del plato en su lugar.
  • No encender el microondas cuando esté vacío.

Para evitar el sobrecalentamiento de los objetos

  • No usar recipientes de cuello estrecho para calentar líquidos.
  • Siempre revisar la temperatura de los alimentos para niños antes de servir.
  • Usar siempre manoplas o guantes de cocina cuando se toquen recipientes calientes.

Descripción del panel de control

Modo microondas

  • Presionar el botón con el ícono de microondas.
  • Seleccionar la potencia deseada presionando varias veces el mismo botón o girando la perilla.
  • Ajustar el tiempo de cocción girando la perilla.
  • Presionar el botón de inicio para comenzar la cocción.

Modo grill

  • Presionar el botón con el ícono de grill.
  • Ajustar el tiempo de cocción girando la perilla.
  • Presionar el botón de inicio para comenzar la cocción.

Modo combi grill

  • Presionar el botón combi grill.
  • Seleccionar el tipo de cocción deseado presionando varias veces el botón o girando la perilla.
  • Ajustar el tiempo de cocción girando la perilla.
  • Presionar el botón de inicio para comenzar la cocción.

Descongelamiento automático

  • Presionar el botón de descongelamiento automático.
  • Seleccionar la categoría de alimento (carne, verduras, pescado, pollo) presionando varias veces el botón o girando la perilla.
  • Ajustar el peso del alimento girando la perilla.
  • Presionar inicio para comenzar la función.

Mantenimiento y limpieza

  • Limpie regularmente el horno, eliminando todos los restos de comida.
  • Usar un paño suave y un detergente suave para las superficies internas y externas.
  • No usar dispositivos de vapor para limpiar el horno.
  • Limpie regularmente el plato giratorio y el soporte del plato.
