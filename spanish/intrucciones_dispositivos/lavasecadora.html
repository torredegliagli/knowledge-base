Instrucciones lavasecadora

Seguridad

  • No utilices el aparato si la superficie está dañada.
  • No coloques objetos sobre la lavasecadora.
  • Supervisa el proceso de lavado y secado, especialmente si hay niños presentes.

Cómo usar la lavasecadora

  • Selección de programa: selecciona el programa deseado mediante el mando A.
    • La lavasecadora dispone de tres modos de funcionamiento: lavado, secado y lavado y secado automáticos. Para una descripción más detallada de los programas, consulta los apartados correspondientes.
  • Ajuste de configuraciones: si es necesario, ajusta la temperatura y el centrifugado en la pantalla pulsando los botones F y G.
  • Detergente: coloca el detergente en el compartimento izquierdo y el suavizante, si se usa, en el central. El compartimento derecho solo debe llenarse para programas con prelavado, dejando vacío el compartimento izquierdo.
  • Inicio: pulsa el botón B de encendido durante aproximadamente 1 segundo.
  • Apagado: cuando el ciclo haya terminado, gira el mando a OFF para abrir la puerta.

Programas de lavado

  • Seleccionando uno de estos programas mediante el mando, se activará un ciclo de solo lavado. El secado puede seleccionarse después y realizarse manualmente, al terminar el ciclo de lavado, seleccionando un programa de secado mediante el mando, o puede realizarse automáticamente, pulsando el botón D en la pantalla de la lavasecadora, después de haber seleccionado un programa de lavado.
  • Algodón: para ropa de algodón resistente.
  • Eco 40-60: para ropa de algodón con suciedad normal.
  • Sintéticos: para ropa de tejidos sintéticos.

Programas de lavado rápido

  • Seleccionando uno de estos programas mediante el mando, se activará un ciclo de solo lavado rápido. El secado puede seleccionarse después y realizarse manualmente, seleccionando un programa de secado mediante el mando cuando el ciclo de lavado haya terminado, o puede realizarse automáticamente, pulsando el botón D en la pantalla de la lavasecadora, después de haber seleccionado cualquier programa de lavado.
  • Rápido 13'/30'/44': lavado adecuado para todo tipo de tejidos. Recomendado para pequeñas cargas y prendas poco sucias. . Pulsa el botón E en la pantalla de la lavasecadora para cambiar la duración del lavado.
  • Mixtos y colores 59': para tejidos mixtos de diferentes colores con suciedad ligera.
  • Algodón 59': para prendas de algodón con suciedad moderada. Para obtener mejores resultados, se recomienda media carga.
  • Special 49': garantiza altas prestaciones de lavado a temperatura media en solo 49 minutos. Para obtener mejores resultados, se recomienda media carga.

Programas de secado

  • Seleccionando uno de estos programas mediante el mando, se activará un programa que incluye un ciclo solo de secado. Utiliza este programa después de un programa de lavado para secar tus prendas. Elige el programa y el nivel de secado deseado, como se explica a continuación.
  • Selecciona el programa de secado mediante el mando:
    • Lana: secado a baja temperatura, ideal para prendas de lana. Se recomienda dar la vuelta a las prendas antes de secarlas.
    • Mixtos: secado a baja temperatura, recomendado para tejidos mixtos y sintéticos.
    • Algodón: secado a alta temperatura, recomendado para tejidos de algodón, toallas, lino y cáñamo.
  • Selecciona el nivel de secado o un secado por tiempo, presionando el botón D en la pantalla de la lavadora-secadora, hasta que veas aparecer en la pantalla el nivel de secado deseado:
    • Extra seco: recomendado para toallas, albornoces y cargas voluminosas.
    • Listo para plancha: deja la ropa lista para planchar.
    • Armario: para prendas que no necesitan planchado.
    • Tiempo: 120'/90'/60'/30'.

Programas de lavado y secado

  • Para realizar un ciclo de lavado y uno de secado de forma automática, selecciona un programa de lavado o de lavado rápido y presiona el botón D en la pantalla de la lavadora-secadora para seleccionar el nivel de secado deseado.
  • Selección del programa de lavado: selecciona el programa de lavado mediante el mando.
  • Nivel de secado: selecciona el nivel de secado o un secado por tiempo, presionando el botón D en la pantalla de la lavadora-secadora hasta que veas aparecer el nivel de secado deseado:
    • Extra seco: recomendado para toallas, albornoces y cargas voluminosas.
    • Listo para plancha: deja la ropa lista para planchar.
    • Armario: para prendas que no necesitan planchado.
    • Tiempo: 120'/90'/60'/30'.

Funciones especiales

  • Nivel de suciedad: esta opción permite elegir entre tres niveles de intensidad de lavado, según el grado de suciedad de los tejidos, modificando la duración del programa seleccionado (puede ser utilizada solo en algunos programas, como se indica en la tabla de programas).
  • Para usar esta función, presiona el botón E en la pantalla de la lavadora-secadora, después de seleccionar un programa de lavado, hasta que veas aparecer el nivel de secado deseado.
  • Protección infantil: presiona simultáneamente los botones F y G durante unos 3 segundos.
