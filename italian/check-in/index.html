Istruzioni per il Check-in

Check-in Automatico (tramite sistema Ok Home)

  • Ricezione Accesso: Riceverai un link per accedere a una web app e un codice di backup per l'uso offline.
  • Accesso: Utilizza il link fornito per accedere alla web app.
  • Controllo delle Porte: Dopo aver effettuato l'accesso, vedrai le icone per tre porte: Porta del Palazzo, Porta del Pianerottolo e Porta d'Ingresso.
  • Apertura delle Porte: Clicca sull'icona corrispondente per aprire la porta desiderata.
  • Nota: La Porta del Palazzo impiega 2-3 secondi per rispondere dopo il clic sullo smartphone. Non cliccare troppo velocemente tra la Porta del Pianerottolo e la Porta d'Ingresso. Utilizzare l'app solo in prossimità delle porte.

Check-in Offline (se il sistema Ok Home non è disponibile)

  • Porta del Palazzo: Suona il campanello "Nardoni" X volte. Questo aprirà la Porta del Palazzo.
  • Accesso alle Chiavi: Una volta al 6° piano, utilizza il tastierino numerico per aprire il bussolotto delle chiavi vicino alla Porta del Pianerottolo.
  • Porta del Pianerottolo: Utilizza le chiavi dal bussolotto per aprire la Porta del Pianerottolo. Riponi le chiavi nel bussolotto dopo l'uso.
  • Porta d'Ingresso: Utilizza il secondo tastierino numerico e inserisci il codice Ok Home fornito al momento della prenotazione per aprire la Porta d'Ingresso.

Note Importanti

  • Il metodo offline prevede alcuni passaggi aggiuntivi, mentre la web app è più intuitiva e veloce, richiedendo solo un tocco sullo schermo per aprire la porta corrispondente. Consigliamo l'uso della web app rispetto al metodo offline.
